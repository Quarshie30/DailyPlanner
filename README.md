## Introduction

The Work Day Scheduler is a web application that provides a user-friendly interface for scheduling daily tasks within standard business hours. It allows users to enter events for each hour, and the app dynamically updates the display based on past, present, and future time blocks.

Work Day Planner is an application that helps users to plan their day to day activities within a typical business day. The Planner allows users to enter tasks for each hour in the business day. Upon entering the app dynamically updates colour coding past, present and future wihtin the blocks.

## Features

- Current day displayed at the top of page
- Enter evens on hourly
- Color-coded time blocks depending on event (past, present, and future)
- Events saved on local storage

## Usage

1. Open the index.html file in a web browser.
2. View and interact with the calendar.
3. Click on a time block to enter an event.
4. Click the save button to save the event.
5. Refresh the page, and the saved events will persist.
